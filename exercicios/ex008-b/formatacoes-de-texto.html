Formatação de textos

Para negrito Use a tag <strong>

Para texto itálico Use a tag <em>

Para texto marcado use a tag <mark>

Para texto pequeno use a tag <small>

Para texto taxado use a tag <del>

Para texto sublinhado use a tag <ins>

Para texto subscrito use a tag <sub>

Para texto acima use a tag <sup>

Para inserir códigos use a tag <code>

Para texto de

citações
use a tag <blockquote>

Para texto abreviado HTML use a tag <abbr>
